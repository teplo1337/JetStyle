```
Задание
Необходимо написать веб-версию экранной клавиатуры.
Клавиатура должна содержать:
все буквы русского алфавита,
все буквы английского алфавита,
кнопку переключения языка между алфавитами,
кнопку удаления предыдущего символа,
пробел,
цифры от 0 до 9.

Навигация по клавиатуре должна осуществляться через tab/shift+tab. Ввод символа через клик левой кнопки мыши, либо по нажатию клавиши enter. Выбранные символы должны быть видны на экране.

Требования:
необходимо использовать синтаксис es6-8, использование JQuery или других аналогичных библиотек не допускается, 
реализация должна быть реиспользуемая. То есть, чтобы заинитить клавиатуру должно быть достаточно в html создать пустой dom-element с каким-то классом,
плюсом будет подумать над реализацией добавления других алфавитов в клавиатуру с минимальными временными затратами.

Пример отображения аналогичной клавиатуры https://monosnap.com/file/a9wGEUSqvzlOB0csasETCSyS45nKqX 

```
